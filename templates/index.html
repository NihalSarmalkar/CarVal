About CarVal

CarVal is the platform where people can check the valuation of their car and can find the true price of their car

CarVal platform is based on the Random Forest machine learning alorithm. Where with the help of user's input algorithm predict the selling price of the car with the accuracy of 91% . This platform is build on Frontend -(HTML,CSS,JS) Backend - (Machine Learning Model,FLASK(Python)) .Data set used for this project was taken from Kaggel.All the source code of the project is being provide on Github.
